# Dr Croissant - GitHub Signature

Pacote pronto para subir no GitHub Pages.

## Arquivos

- `index.html`
  Preview responsivo com a arte inteira e links por area.
- `signature-hosted.html`
  Versao montada com fatias em alta resolucao.
- `assets/assinatura_full.png`
  PNG original aprovado.
- `assets/sig_*.png`
  Fatias usadas na versao hospedada.

## Publicacao

1. Crie um repositorio no GitHub.
2. Envie todos os arquivos desta pasta.
3. Ative o GitHub Pages apontando para a branch principal.
4. Use a URL publicada para abrir:
   - `index.html`
   - `signature-hosted.html`
